利用規約

このツール（以下「本サービス」）は、YouTube API を利用して急成長しているチャンネル情報を検索・分析する目的で提供される非公式ツールです。

第1条（利用の条件）

本サービスの利用にあたり、ユーザーはYouTube API利用規約および本規約に同意したものとみなします。

第2条（YouTube APIの利用）

本サービスは Google および YouTube とは一切関係がなく、Google の API 使用ポリシーに基づいて独自に運営されています。

第3条（知的財産）

表示されるサムネイルやチャンネル情報等の著作権は各提供元に帰属します。

第4条（免責事項）

本サービスの内容・精度・可用性について一切の保証を行いません。

第5条（変更・終了）

本サービスは予告なく内容の変更・停止・終了されることがあります。
